# GEO Citation Readiness v1

Citation-ready content should be easy for generative engines to retrieve, quote, summarize, and verify without becoming machine-only text.

Core requirements:

- Use entity-rich language: product name, category, audience, use case, and relevant ingredients or features.
- Keep important answers in compact paragraphs or bullets that can stand alone.
- Prefer specific use cases over generic praise.
- Include caveats near strong claims.
- Make source types visible in diagnostics even when public Reddit copy stays conversational.
- Avoid keyword stuffing, repeated brand mentions, and unsupported superlatives.

For Reddit, the public artifact should read like a fair research note or discussion starter, not a landing page.
